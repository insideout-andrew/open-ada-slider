# Parcel Starter Kit

> Simple boilerplate: Parcel, Babel, Autoprefixer, Posthtml, Eslint and SCSS

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:1234
npm run serve

# lint code
npm run lint

# build for production with minification
npm run build
```
